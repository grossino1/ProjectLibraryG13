\doxysubsection{Detailed Description}
Rappresenta l\textquotesingle{}utente con privilegi amministrativi (Librarian). 

Questa classe gestisce le credenziali dell\textquotesingle{}amministratore del sistema. I campi sono immutabili ({\ttfamily final}) per garantire che, una volta istanziato il bibliotecario, le sue credenziali non possano essere alterate a runtime.

\begin{DoxyInvariant}{Invariant}
username != null \&\& !username.is\+Empty() (Le credenziali devono essere valide). 

password != null \&\& !password.is\+Empty().
\end{DoxyInvariant}

\doxysubsection{Detailed Description}
Rappresenta l\textquotesingle{}entità prestito all\textquotesingle{}interno del sistema di gestione bibliotecaria. 

Il prestito nasce dall\textquotesingle{}associazione di un libro ed un utente per un periodo di tempo limitato.

\doxysubsection{Detailed Description}
Rappresenta un utente registrato nel sistema bibliotecario. 

\begin{DoxyAuthor}{Author}
chiara
\end{DoxyAuthor}


Questa classe gestisce i dati anagrafici dell\textquotesingle{}utente e mantiene lo stato attuale dei suoi prestiti.

\begin{DoxyInvariant}{Invariant}
matricola != null (Ogni utente deve essere identificabile univocamente). 

lista\+Prestiti != null (La lista prestiti deve essere inizializzata, anche se vuota).
\end{DoxyInvariant}

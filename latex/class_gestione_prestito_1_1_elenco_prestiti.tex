\doxysubsection{Detailed Description}
Gestisce l\textquotesingle{}insieme di tutti i prestiti attivi. 

Questa classe agisce come contenitore principale per gli oggetti \doxylink{class_gestione_prestito_1_1_prestito}{Prestito}. Utilizza un {\ttfamily Tree\+Set} per garantire che non ci siano duplicati. Delega le regole di validazione (numero minimo di copie e numero massimo di prestiti per utente) alla classe \doxylink{class_gestione_prestito_1_1_gestore_prestito}{Gestore\+Prestito}.

\begin{DoxyInvariant}{Invariant}
elenco\+Prestiti != null (La struttura dati è sempre inizializzata). 

gestore != null (La classe deve sempre avere un riferimento valido al gestore delle regole).
\end{DoxyInvariant}

\begin{DoxyEnumerate}

\item Delega al \doxylink{class_gestione_prestito_1_1_gestore_prestito}{Gestore\+Prestito} l\textquotesingle{}aggiornamento dei dati.
\item Salva l\textquotesingle{}elenco aggiornato sul file.
\end{DoxyEnumerate}

\begin{DoxyPostcond}{Postcondition}

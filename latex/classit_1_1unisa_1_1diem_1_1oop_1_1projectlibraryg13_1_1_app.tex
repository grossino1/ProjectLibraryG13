\doxysubsection{Detailed Description}
Classe principale (Entry Point) dell\textquotesingle{}applicazione gestionale. 

Questa classe estende {\ttfamily Application} di Java\+FX ed è responsabile dell\textquotesingle{}avvio dell\textquotesingle{}interfaccia grafica. Si occupa di inizializzare lo Stage primario (finestra), caricare la vista iniziale (Login) e impostare le configurazioni di base (titolo, icona).

Metodo di avvio del ciclo di vita Java\+FX. 

Viene chiamato automaticamente dal framework dopo l\textquotesingle{}inizializzazione del sistema. Esegue i seguenti passaggi\+:
\begin{DoxyEnumerate}
\item Carica il file FXML della vista Login.
\item Configura la Scena e lo Stage.
\item Rende visibile la finestra.
\end{DoxyEnumerate}

\begin{DoxyPrecond}{Precondition}
primary\+Stage != null (Il framework garantisce che lo stage primario sia istanziato). 

Il file FXML "{}/\+GUI/\+GUI\+\_\+\+Login/\+Login\+View.\+fxml"{} deve esistere nelle risorse. 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
La finestra dell\textquotesingle{}applicazione è visibile a schermo.
\end{DoxyPostcond}

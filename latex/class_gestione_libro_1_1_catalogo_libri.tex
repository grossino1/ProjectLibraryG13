\doxysubsection{Detailed Description}
Gestisce l\textquotesingle{}insieme dei libri presenti nel sistema bibliotecario. 

Questa classe funge da container principale per gli oggetti di tipo \doxylink{class_gestione_libro_1_1_libro}{Libro}. Utilizza internamente un {\ttfamily Tree\+Set} per mantenere i libri ordinati secondo il loro ordinamento naturale (definito in \doxylink{class_gestione_libro_1_1_libro_ada9d5af6b64a175f603a5f94eefe4fcc}{Libro.\+compare\+To}) ed evitare duplicati.

\begin{DoxyInvariant}{Invariant}
catalogo\+Libri != null (Il riferimento alla collezione interna non è mai nullo). 

catalogo\+Libri non contiene elementi null (Tree\+Set non accetta null).
\end{DoxyInvariant}

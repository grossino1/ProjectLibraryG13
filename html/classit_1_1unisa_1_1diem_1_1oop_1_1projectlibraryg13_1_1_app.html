Detailed Description

Classe principale (Entry Point) dell'applicazione gestionale.

Questa classe estende Application di JavaFX ed è responsabile dell'avvio dell'interfaccia grafica. Si occupa di inizializzare lo Stage primario (finestra), caricare la vista iniziale (Login) e impostare le configurazioni di base (titolo, icona).

void it.unisa.diem.oop.projectlibraryg13.App.start ( Stage primaryStage)
inline

Metodo di avvio del ciclo di vita JavaFX.

Viene chiamato automaticamente dal framework dopo l'inizializzazione del sistema. Esegue i seguenti passaggi:

  1. Carica il file FXML della vista Login.
  2. Configura la Scena e lo Stage.
  3. Rende visibile la finestra.
Precondition
primaryStage != null (Il framework garantisce che lo stage primario sia istanziato).
Il file FXML "/GUI/GUI_Login/LoginView.fxml" deve esistere nelle risorse.
Postcondition
La finestra dell'applicazione è visibile a schermo.
Parameters
[in]primaryStageIl contenitore principale della finestra (fornito dal framework).

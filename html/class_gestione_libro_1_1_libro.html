Detailed Description

Rappresenta l'entità libro all'interno del sistema di gestione bibliotecaria.

Questa classe gestisce i dati del libro e mantiene lo stato attuale del numero di copie ancora disponibili.

Invariant
numeroCopie >= 0 (Il numero di copie non può mai essere negativo).
isbn != null (Ogni libro deve avere un codice ISBN).

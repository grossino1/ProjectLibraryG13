Detailed Description

Rappresenta un utente registrato nel sistema bibliotecario.

Author
chiara

Questa classe gestisce i dati anagrafici dell'utente e mantiene lo stato attuale dei suoi prestiti.

Invariant
matricola != null (Ogni utente deve essere identificabile univocamente).
listaPrestiti != null (La lista prestiti deve essere inizializzata, anche se vuota).
See also
GestionePrestito.Prestito
java.io.Serializable
Author

Confronta questo utente con un altro in base all'ordinamento alfabetico. L'ordinamento avviene prima per cognome e, in caso di parità, per nome. Nel caso in cui due utenti hanno lo stesso cognome e nome, allora si ordina per matricola.

Parameters

Returns

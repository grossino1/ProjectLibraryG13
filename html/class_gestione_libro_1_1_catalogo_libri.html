Detailed Description

Gestisce l'insieme dei libri presenti nel sistema bibliotecario.

Questa classe funge da container principale per gli oggetti di tipo Libro. Utilizza internamente un TreeSet per mantenere i libri ordinati secondo il loro ordinamento naturale (definito in Libro.compareTo) ed evitare duplicati.

Invariant
catalogoLibri != null (Il riferimento alla collezione interna non è mai nullo).
catalogoLibri non contiene elementi null (TreeSet non accetta null).

Registra un nuovo libro nel catalogo.

Aggiunge un libro alla collezione. Se il libro è già presente (stesso ISBN), il Set non lo duplicherà (rispettando l'equals di Libro), ma non verrà lanciato errore. L'errore viene lanciato solo se l'ISBN non è valido strutturalmente.

Precondition
l != null (Non si possono inserire libri nulli).
Postcondition
catalogoLibri.contains(l) == true.
size() >= old_size().
Parameters
[in]lL'oggetto Libro da aggiungere.
Exceptions
ISBNNotValidExceptionSe il libro ha un formato ISBN non valido.
LibroNotFoundExceptionSe il libro non è presente nel catalogo.
CatalogoPienoExceptionSe il catalogo ha raggiunto la capienza massima.
IOExceptionSe il salvataggio fallisce.
LibroPresenteExceptionSe il libro è già presente nel catalogo.
